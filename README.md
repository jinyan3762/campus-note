# 校园记 (CampusNote)

> 专为大学生设计的一站式本地效率工具

## 项目简介

"校园记"是一款面向大学生群体的本地化桌面应用，集成了校园生活中最常用的六大功能模块。它解决了以下核心痛点：

- **交流需求**：通过内置校园论坛，学生可在本地网络环境内自由发帖、评论，形成紧密的校园社区。
- **日程管理**：提供课表查询、记事本和待办事项三大工具，覆盖从课程安排到日常笔记的全方位时间管理。
- **决策困难**：创新的"转盘决策"功能，帮助用户从待办事项中随机抽取一项执行，告别选择困难症。
- **社交匹配**：基于兴趣标签的搭子匹配系统，帮助学生找到志同道合的伙伴一起完成任务。
- **数据统计**：新增学习数据仪表盘，可视化展示任务完成率、活跃天数等个人数据指标。

本项目完全本地运行，数据存储于 SQLite，无需联网，无需安装额外数据库或服务。

## 功能说明

### 模块一：用户系统
- 本地注册与登录，密码采用 SHA-256 加密存储
- 登录后进入主控面板，展示各功能模块入口和统计数据
- 支持编辑个人资料（昵称、兴趣标签）

### 模块二：校园论坛 (BBS)
- 浏览所有帖子列表（标题、作者、发布时间）
- 发布新帖子（标题 + 正文内容）
- 查看帖子详情，支持评论互动
- 作者可删除自己发布的帖子

### 模块三：校园记事本
- 便利贴风格的笔记卡片（多彩马卡龙色系背景）
- 支持增加、编辑、删除笔记
- 笔记按修改时间倒序排列
- 支持关键词搜索功能

### 模块四：课表查询
- 5×7 可视化课表网格（周一至周日 × 第1-5节课）
- 点击空格添加课程（课程名 + 教室）
- 支持编辑和删除已添加的课程
- 数据持久化存储，每人独立课表

### 模块五：待办事项 + 转盘决策（核心亮点）
- **普通模式**：添加/删除待办事项，标记完成/未完成
- **转盘模式**：
  - Canvas 绘制的彩色马卡龙转盘，每个扇区对应一个待办事项
  - 点击"随机抽取"，转盘带动画旋转并随机停止
  - 旋转结束高亮显示选中的待办事项，并滚动定位
  - **搭子联动**：每个待办事项旁有"找搭子做"按钮，点击自动跳转匹配模块

### 模块六：搭子匹配
- 个人资料卡（可编辑昵称、兴趣标签如"学习""运动""游戏""吃饭"等）
- **匹配算法**：综合兴趣标签重合度（Jaccard 相似度，权重 70%）+ 最近活跃度（权重 30%），按匹配度降序展示前 3 名用户
- 支持向匹配到的用户发送"组队邀请"，可接受/拒绝收到的邀请
- 从待办事项跳转时自动触发匹配

### 模块七：学习数据仪表盘（新增）
- 可视化展示每日/每周任务完成率
- 统计活跃天数、连续打卡记录
- 饼图展示各类型任务占比
- 生成学习周报，支持导出

## 运行方法

### 第一步：安装 Python

本应用需要 **Python 3.7 或更高版本**（推荐 3.8+）。

**Windows 用户：**

1. 打开 [Python 官网](https://www.python.org/downloads/) 下载安装包（推荐 3.10+）
2. 运行安装包，**务必勾选底部的 "Add Python to PATH"** 复选框
3. 点击 "Install Now" 完成安装
4. 打开 **PowerShell**（Win + R → 输入 `powershell`），验证安装：

   ```powershell
   python --version
   ```

   如果显示 `Python 3.x.x` 即表示安装成功。

> ⚠️ **最常见的问题：`python` 命令打开了 Microsoft Store**
>
> 如果在 PowerShell 中输入 `python` 后跳转到微软应用商店，说明 Windows 的「应用执行别名」拦截了该命令。解决方法：
>
> 1. 打开 **Windows 设置** → **应用** → **应用执行别名**（或直接搜索"应用执行别名"）
> 2. 将 **"python.exe"** 和 **"python3.exe"** 的开关 **全部关闭**
> 3. 重新打开 PowerShell，再次输入 `python --version` 验证
>
> 如仍不行，请重新运行 Python 安装包，确保勾选 "Add Python to PATH"（建议勾选后选择 "Customize installation" → 确认 "Add Python to environment variables" 已勾选）。

**macOS 用户：**

```bash
# 使用 Homebrew 安装
brew install python3
```

### 第二步：下载项目

```powershell
# 克隆仓库（在 PowerShell 中执行）
git clone https://github.com/jinyan3762/campus-note.git
cd campus-note
```

> 如果没有安装 Git，可直接在 GitHub 页面点击 **"Code → Download ZIP"**，解压到本地文件夹后进入该目录。

---

### 第三步：安装依赖

在项目根目录下打开 **PowerShell**（或终端），执行：

```powershell
python -m pip install -r requirements.txt
```

> 💡 使用 `python -m pip` 而非直接 `pip`——即使 `pip` 未加入 PATH 也能正常运行。

如果下载速度慢，可使用国内镜像：

```powershell
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**依赖项：**

| 包 | 版本 | 用途 |
|---|---|---|
| Flask | ≥2.2, <3.0 | Web 框架（兼容 Python 3.7+） |
| Werkzeug | ≥2.2, <3.0 | WSGI 工具库 |

> 旧版 `requirements.txt` 指定了 `Flask==3.0.0`，该版本不兼容 Python 3.7；现已修正为兼容范围。如果你之前安装失败，请更新项目后重试。

---

### 第四步：启动应用

在项目根目录的 PowerShell 窗口中执行：

```powershell
python app.py
```

启动成功后，终端会显示：

```
==================================================
  校园记 (CampusNote) 已启动!
  浏览器访问: http://127.0.0.1:5000
  默认账号: admin / 123456
==================================================
 * Running on http://127.0.0.1:5000
```

> ⚠️ **注意**：
> - 这是本地 Web 应用，需要浏览器访问，不会弹出桌面窗口
> - **不要关闭这个 PowerShell 窗口**，关闭即停止服务
> - 首次运行时，Windows 防火墙可能弹出安全提示，请点击 **"允许访问"**
> - 如果 Windows 系统主机名包含中文，应用已内置编码兼容修复，不会因此崩溃

### 常见问题排查

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| `python` 跳转到 Microsoft Store | Windows 应用执行别名拦截 | **设置 → 应用 → 应用执行别名 → 关闭 python.exe 开关**（详见第一步） |
| `python --version` 无反应 | Python 未安装或 PATH 未生效 | 重启电脑后再试；或卸载重装 Python 并勾选 "Add Python to PATH" |
| `pip install` 报连接错误 | 网络问题，PyPI 被墙 | 使用国内镜像：`-i https://pypi.tuna.tsinghua.edu.cn/simple` |
| `pip install` 报版本不兼容 | Python 版本过旧 | 确认 `python --version` ≥ 3.7；如为 3.6 请升级 Python |
| `ModuleNotFoundError: No module named 'flask'` | 依赖未安装 | 重新执行第三步 `python -m pip install -r requirements.txt` |
| 启动报 `UnicodeDecodeError` | 旧版代码未包含编码修复 | 更新到最新代码（`git pull`），v1.0.3 已内置修复 |
| 浏览器访问 `127.0.0.1:5000` 无响应 | 应用未启动或端口被占用 | 确认终端显示 "Running on..."；检查防火墙是否拦截 |
| 端口 5000 被占用 | 其他程序占用了 5000 端口 | 修改 `app.py` 最后一行的 `port=5000` 为其他端口如 `port=8080` |
| 中文终端显示乱码 | Windows 控制台编码问题 | 不影响功能使用；推荐使用 **Windows Terminal** |
| 页面空白/样式丢失 | 浏览器缓存 | 按 `Ctrl + F5` 强制刷新页面 |
| 登录后页面不跳转 | JavaScript 被禁用 | 确保浏览器未禁用 JavaScript |

## AI 使用说明

本项目由 **Claude Code**（Anthropic 的 AI 编程助手）辅助生成。

### 提示词设计思路

项目构建指令（`校园记.md`）遵循以下设计原则：

1. **结构化需求描述**：将项目需求分为技术栈、功能模块、Git 操作和文档四大板块，确保 AI 能准确理解全貌。
2. **模块化拆分**：六大功能模块各自独立描述，附带具体的交互细节和存储要求，便于 AI 分步实现。
3. **明确的约束条件**：指定 UI 风格（马卡龙色系）、技术栈（Flask + 原生前端）和运行方式，减少歧义。
4. **分步提交策略**：要求 5 次有意义的 Git Commit，使开发过程可追溯。

### Claude Code 在关键功能中的作用

1. **Canvas 转盘绘制与动画**：
   - 使用 HTML5 Canvas API 绘制环形转盘，每个扇区着色为不同的马卡龙色系
   - 通过 `requestAnimationFrame` 实现平滑的缓出（ease-out）旋转动画
   - 随机目标角度计算确保结果不可预测
   - 旋转结束后高亮结果并自动滚动定位

2. **搭子匹配算法**：
   - 基于 Jaccard 相似度计算兴趣标签重合度
   - 综合活跃度因子（在线状态 + 最近活跃时间）进行加权排序
   - 返回匹配度最高的前 3 名用户

3. **SPA 前端路由**：
   - 基于 URL Hash 的单页应用路由，无需刷新页面即可切换模块
   - 未登录拦截重定向
